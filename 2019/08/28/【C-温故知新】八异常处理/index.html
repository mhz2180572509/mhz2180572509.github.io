异常的抛出与捕获

异常是程序在执行期间产生的问题，比如尝试除以零的操作。异常提供了一种转移程序控制权的方式。C++ 异常处理涉及到三个关键字：try、catch、throw。

  • throw：当问题出现时，程序抛出一个异常
  • catch：在想要处理问题的地方，通过异常处理程序捕获异常
  • try：try块中的代码标识将被激活的特定异常。它后面通常跟着一个或多个 catch 块。
